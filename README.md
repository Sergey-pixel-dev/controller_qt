# DM25_Controller_PC

Десктопное приложение на Qt/C++ для управления и мониторинга высоковольтного импульсного блока (до 30 кВ). Работает поверх Modbus RTU.

## Что делает

- Управление импульсным генератором по Modbus RTU через UART на 2 Мбод: частота, длительность, дискретные выходы (накал, У.Э., -25 кВ, HE, LE)
- Захват и визуализация сигналов АЦП в реальном времени на Qt Charts с настраиваемыми масштабами (мкс/дел, В/дел)
- Стробоскопическая эквивалентно-временная выборка (26 кадров x 24 отсчёта) для восстановления быстрых импульсов при ограниченной частоте дискретизации
- Демультиплексирование двух протоколов в одном UART-потоке: Modbus RTU (управление) и бинарный поток данных АЦП (заголовок `0xAA55`, ~2968 байт/пакет)
- Цифровая обработка сигналов: FIR-фильтр (Blackman-окно, 101 коэффициент), скользящее среднее, пороговый фильтр

## Стек

C++, Qt (Widgets), Modbus RTU, UART (Serial Port), многопоточность (QThread)
